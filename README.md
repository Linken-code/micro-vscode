# micro-vscode

A lightweight code editing and submission tool. The micro-vscode integrated the basic function of git.

This project is only used for learning.

## Development

### In Browser

open http://localhost:1420/

```bash
pnpm run dev
```

### In Tauri Window

```bash
pnpm tauri dev
```

### Build

```bash
pnpm tauri build
```
